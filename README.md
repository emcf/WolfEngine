# WolfEngine

In this project, I made a hacky 3D Engine using only the System.Drawing library. That means _No OpenGL, no OpenTK, no DirectX, no XNA._

The engine includes:
* Depth Buffer Lighting
* Textures
* Easy Map Creation
* Sound (which needs some work)
